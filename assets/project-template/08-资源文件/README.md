# 资源文件目录

> **存放项目相关资源文件**

---

## 📂 目录结构

```
08-资源文件/
├── api/              # API相关文件
│   └── swagger.json  # Swagger导出文件
├── images/           # 图片资源
│   ├── architecture/ # 架构图
│   └── screenshots/  # 功能截图
└── other/            # 其他资源
```

---

## 📁 各目录说明

### api/
**用途**：存放API相关文件
- `swagger.json` - Swagger自动导出的API文档
- `postman_collection.json` - Postman测试集合

**更新频率**：API变更后及时导出

---

### images/
**用途**：存放图片资源

#### architecture/
- 系统架构图
- 数据库ER图
- 流程图

#### screenshots/
- 功能截图
- 界面原型图
- 效果图

**文件格式**：PNG / JPG / SVG

---

### other/
**用途**：其他类型的资源文件
- PDF文档
- Excel文件
- 其他格式文件

---

## 📝 使用规范

### 1. 文件命名
- 使用小写字母+下划线
- 包含日期或版本号
- 示例：`system_architecture_v1.0.png`

### 2. 文件大小
- 图片尽量压缩
- 单个文件不超过5MB

### 3. 版本管理
- 重要文件保留版本号
- 旧版本可以保留备份
